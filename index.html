Spotify-Local: A multi-platform API to control the local Spotify Client¶

Spotify-Local library is designed to make controlling the Spotify client on your local machine possible! This is a wrapper for the web helper process which exposes a simple api. Spotify-Local-Control is inspired by SpotifyAPI-NET. This library allows you to perform simple actions quickly, or listen to events and register callbacks when a song changes, or the pause button is pushed.

When using this library you automatically get:

  • The ability to play/pause the current song
  • The ability to change tracks
  • You can register callbacks and listen for events when the state of Spotify changes
  • A nice context manager api using with

Installation¶

$ pipenv install spotify-local-control

Only Python 3.6 is supported.

Tutorial & Usage¶

Connect to the Spotify Client (Spotify must be open to do this):

>>> from spotify_local import SpotifyLocal

>>> with SpotifyLocal() as s:
        pass

Pause the Spotify Client:

>>> with SpotifyLocal() as s:
        s.pause()

Grab the current state of the Spotify client, including now playing information:

>>> with SpotifyLocal() as s:
        print(s.get_current_status())

Play a playlist, song, album, artist, etc using a Spotify uri link:

>>> with SpotifyLocal() as s:
        s.playURI('spotify:track:0thLhIqWsqqycEqFONOyhu')

Register a callback and listen for events:

>>> from spotify_local import SpotifyLocal
>>> def test(event):
...     print(event)
>>> s = SpotifyLocal()
>>> s.connect()
>>> s.on_status_change += test
>>> s.listen_for_events()
>>> print("Do more stuff because that runs in the background")

Main Classes¶

These classes are the main interface to spotify_local:

class spotify_local.SpotifyLocal[source]¶

A basic controller for the local Spotify client.

connect() → None[source]¶

Register the oauth token, the csrf token, and set connected to true

disconnect() → None[source]¶

Join the process and set the connection to false.

get_current_status() → Mapping[source]¶

Returns the current state of the local spotify client

listen_for_events(wait: int = 60) → None[source]¶

Listen for events and call any associated callbacks when there is an event. This is a non-blocking operation.

pause(pause=True) → None[source]¶

Pauses the spotify player

Parameters:pause – boolean value to choose the pause/play state
playURI(uri: str) → Mapping[source]¶

Play a Spotify uri, for example spotify:track:5Yn8WCB4Dqm8snemB5Mu4K

previous() → None[source]¶

Goes to the beginning of the track, or if called twice goes to the previous track.

skip() → None[source]¶

Skips the current song

unpause() → None[source]¶

Unpauses the player by calling pause()
